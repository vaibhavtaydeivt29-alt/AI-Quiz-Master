# 🧠 AI Quiz Master

An AI-powered quiz generation and evaluation system that creates interactive quizzes, assesses user performance, and provides instant feedback using Generative AI.

---

## 🚀 Overview

AI Quiz Master is designed to automate quiz creation and evaluation. Users can generate quizzes on various topics, answer questions, and receive detailed performance analysis.

The project demonstrates the use of AI Agents, Prompt Engineering, and Generative AI for educational and assessment applications.

---

## ✨ Features

### 📚 Quiz Generation
- Generate quizzes on any topic
- Multiple-choice questions
- Custom difficulty levels
- Dynamic question creation

### 🎯 Quiz Evaluation
- Automatic answer checking
- Score calculation
- Performance tracking

### 📊 Performance Analysis
- Strength and weakness identification
- Detailed feedback
- Learning recommendations

### 🤖 AI-Powered Learning
- Adaptive question generation
- Personalized learning support
- Intelligent assessment

## 📂 Project Workflow

1. User selects a topic
2. AI generates quiz questions
3. User answers questions
4. System evaluates responses
5. Score is calculated
6. Feedback and recommendations are generated
